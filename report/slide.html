Bộ Dữ Liệu GTSRB

📊 German Traffic Sign Recognition Benchmark
  • Tổng cộng: ~50,000 ảnh thuộc 43 lớp phân loại
  • Định dạng: PPM/PNG (RGB), kích thước đa dạng
  • Cấu trúc thư mục theo tốc độ: 0/, 1/, 2/ (20, 30, 50 km/h)
  • Chuyển đổi về 30×30×3 cho phù hợp MobileNetV2 resize
⚠️ Phân Tích Mất Cân Bằng
  • Lớp 0: 1500 ảnh (lớp nhiều nhất)
  • Lớp 37-42: <200 ảnh (lớp ít nhất)
  • Tỷ lệ max/min ≈ 10:1
→ Cần áp dụng Class Weights để xử lý mất cân bằng

Model C: Deeper CNN + BatchNorm

🏆 BEST MODEL
🏗️ Kiến Trúc 3 Blocks Deep CNN
  • Block 1: 30×30 → 15×15 (Double-Conv 32) + BN + MaxPool + Dropout(0.25)
  • Block 2: 15×15 → 7×7 (Double-Conv 64) + BN + MaxPool + Dropout(0.25)
  • Block 3: 7×7 → 3×3 (Double-Conv 128) + BN + MaxPool + Dropout(0.25)
  • Output: Flatten(1152) → Dense(256) + BN + Dropout(0.25) → Dense(43, softmax)

Tổng tham số: ~2.27 MB | Số lớp: 17 layers

🎯
KẾT QUẢ TỐT NHẤT (30 epochs)
99.96%
F1-Score: 0.999 (Macro Avg) / 1.000 (Weighted)
✨ BatchNormalization Benefits
  • Giảm Internal Covariate Shift: Chuẩn hóa activation theo mini-batch
  • Learning Rate cao hơn: Cho phép LR lớn hơn mà không bị explode/vanish
  • Regularization Effect: Thêm noise từ batch statistics giúp chống overfit
  • Huấn luyện nhanh & ổn định hơn: Giảm số epoch cần thiết
  • Khởi tạo ít nhạy cảm: Giảm phụ thuộc vào initialization
💡 So Sánh Với Model A

Model C cải thiện +0.77% so với Baseline CNN nhờ:
• 3 blocks thay vì 2 blocks
• Padding "same" giữ nguyên kích thước feature maps
• BatchNorm sau mỗi Conv2D layer

Model D: Transfer Learning (MobileNetV2)

🔄 Kiến Trúc Transfer Learning
  • Base model: MobileNetV2 pretrained trên ImageNet
  • Input: 96×96×3 (resize từ 30×30)
  • Freeze strategy: Freeze 20 layers đầu, unfreeze 20 layers cuối
  • Preprocessing: Lambda layer chuẩn hóa ảnh về [-1, 1]
  • Pooling: GlobalAveragePooling2D → Vector 1280 chiều
  • Classifier: Dense(256, relu) + Dropout(0.5) + Dense(43, softmax)

Tổng tham số: ~5.89 MB | Trainable: ~1.45 MB

⚙️ Hyperparameters

Learning Rate: 10-4 (thấp hơn 10× so với Adam mặc định)

  • Optimizer: Adam với learning rate thấp
  • Loss: Categorical Crossentropy
  • Epochs: 30 epochs + ReduceLROnPlateau
